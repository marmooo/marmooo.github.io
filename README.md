# @marmooo

[https://marmooo.github.io/](https://marmooo.github.io/)

## Installation

1. install [gitn](https://github.com/marmooo/gitn)
2. `gitn clone vendor repos.lst`
3. `bash install-fonts.sh`

## Build

```
bash build.sh
```
